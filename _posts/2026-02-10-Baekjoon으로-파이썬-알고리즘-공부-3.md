## 5. 문자열

string type 을 다루는 여러 문제를 풀어보자.
우선 문자열이란, 연속된 문자들의 나열이며 python 에서는 "" 또는 '' 를 감싸 표현한다.
{% highlight text %}
"apple"
'3-2'
"Hello world"
{% endhighlight %}

### 문자열 인덱싱

문자열은 행렬과 비슷하게 문자열 속 특정 문자를 인덱싱을 통해 가져올 수 있다.
{% highlight python %}
a = "Hello world"
print(a[0])
> "H"

print(a[6])
> "w"

print(a[-1]) // -1(d), -2(l), -3(r), .. 음수 인덱싱은 뒤에서 부터 차례대로 돌려 받는다.
> "d"
{% endhighlight %}

### 단어 길이 재기

문자열의 길이를 측정하기 위해서는 len 함수를 사용하면 된다.
{% highlight python %}
print(len("pulljima"))
> 8

a = "I am happy."
print(len(a))
> 11
{% endhighlight %}

### 아스키 코드

아스키 코드(ASCII)는 문자, 숫자, 특수문자 등을 나타내는 7비트 문자 인코딩 표준이다. 총 128개의 문자로 구성되어 있으며, 0부터 127까지의 숫자로 표현된다.

![아스키 코드 표](/images/ascii.png)

**sys.stdin.readline()**

{% highlight python %}
// input : hello world
import sys
a = sys.stdin.readline()
print(a)
>> 'hello world\n'
{% endhighlight %}

{% highlight python %}
import sys
a, b = map(int, sys.stdin.readline().split())
{% endhighlight %}

sys.stdin.readline() 이 위에서 언급했듯이 input() 보다 빠른 이유로는 2가지가 있다.
1. prompt message를 받지 않는다. (ex. prompt message는 이런 것이다 >> input(*"입력:"*))
2. 개행문자를 그대로 받아서 반환한다.
