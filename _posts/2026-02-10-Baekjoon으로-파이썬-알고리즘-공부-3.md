## 5. 문자열

string type 을 다루는 여러 문제를 풀어보자.
우선 문자열이란, 연속된 문자들의 나열이며 python 에서는 "" 또는 '' 를 감싸 표현한다.
{% highlight text %}
"apple"
'3-2'
"Hello world"
{% endhighlight %}


### 문자열 인덱싱

문자열은 행렬과 비슷하게 문자열 속 특정 문자를 인덱싱을 통해 가져올 수 있다.
{% highlight python %}
a = "Hello world"
print(a[0])
> "H"

print(a[6])
> "w"

print(a[-1]) // -1(d), -2(l), -3(r), .. 음수 인덱싱은 뒤에서 부터 차례대로 돌려 받는다.
> "d"
{% endhighlight %}


### 단어 길이 재기

문자열의 길이를 측정하기 위해서는 len 함수를 사용하면 된다.
{% highlight python %}
print(len("pulljima"))
> 8

a = "I am happy."
print(len(a))
> 11
{% endhighlight %}


### 아스키 코드

아스키 코드(ASCII)는 문자, 숫자, 특수문자 등을 나타내는 7비트 문자 인코딩 표준이다. 총 128개의 문자로 구성되어 있으며, 0부터 127까지의 숫자로 표현된다.


![아스키 코드 표](/images/ascii.png)


특정 문자를 아스키 코드값으로 변환하기 위해서는 ord 함수를 사용하면 된다.
{% highlight python %}
a = "A"
print(ord(a))
> 65
{% endhighlight %}


### 문자열 리스트 변환

문자열에 list()를 씌우면 리스트로 변환할 수 있다.

{% highlight python %}
a = "hello"
b = list(a)
print(b)
> ['h', 'e', 'l', 'l', 'o']
{% endhighlight %}


### 알파벳 찾기 문제

> 알파벳 소문자로만 이루어진 단어 S가 주어진다. 각각의 알파벳에 대해서, 단어에 포함되어 있는 경우에는 처음 등장하는 위치를, 포함되어 있지 않은 경우에는 -1을 출력하는 프로그램을 작성하시오.

해당 문제를 풀기 위해 알파벳 리스트를 생성하는 방법을 알아보자. 아래와 같은 코드를 통해 lower 에 26개의 알파벳이 순서대로 들어가있는 리스트를 생성할 수 있다.

{% highlight python %}
import string
lower = [i for i in string.ascii_lowercase]
{% endhighlight %}

**답변**
{% highlight python %}
import string

s = input()

lower = [i for i in string.ascii_lowercase] // 알파벳 리스트
num = [-1 for i in range(26)] // 기본 -1 값으로 세팅
for i in range(len(s)):
    for j in range(len(lower)):
        if (s[i] == lower[j]) and (num[j] == -1):
            num[j] = i
for i in num:
    print(i, end=" ")
{% endhighlight %}


### 문자열 출력 (with *)

문자열 출력 시 동일한 문자열에 곱하기를 해주면, 해당 횟수만큼 반복되어 출력할 수 있다.
{% highlight python %}
print("*" * 4)
> ****

print("H"*2 + " "*2 + "O")
> "HH  O"
{% endhighlight %}


### 딕셔너리 자료형

Key 와 Value 쌍으로 이루어진 요소들로 구성되어있다. key 와 value 에는 여러 타입의 값들이 들어갈 수 있으며, 서로 타입이 달라도 된다.

{% highlight python %}
{Key1: Value1, Key2: Value2, Key3: Value3, ...}
{% endhighlight %}

**예시**
{% highlight python %}
a = {1: 'a', 2: 'b', 'name': 'pey', 3: [1, 2, 3]}

a['a'] = 'apple' // 딕셔너리 요소 추가
print(a)
> {1: 'a', 2: 'b', 'name': 'pey', 3: [1, 2, 3], 'a': 'apple'}

print(a[1]) // key 로 value 얻기
> 'a'
{% endhighlight %}
