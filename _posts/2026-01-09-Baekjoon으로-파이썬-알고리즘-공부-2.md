## 3. 반복문

for과 while문을 사용하여 해결하는 문제들로 구성되어있다.


### 빠른 A+B

백준 문제를 풀 때, 여러 줄에 걸쳐 들어오는 입력값을 input() 함수와 for문을 이용해 해결했었다. 빠른 A+B 문제에서는 여러 번 반복되는 input()의 경우
timeout을 유발할 수 있기에 input() 대신 sys.stdin.readline() 을 사용해 문제를 풀도록 한다.

**sys.stdin.readline()**

{% highlight python %}
// input : hello world
import sys
a = sys.stdin.readline()
print(a)
>> 'hello world\n'
{% endhighlight %}

{% highlight python %}
import sys
a, b = map(int, sys.stdin.readline().split())
{% endhighlight %}

sys.stdin.readline() 이 위에서 언급했듯이 input() 보다 빠른 이유로는 2가지가 있다.
1. prompt message를 받지 않는다. (ex. prompt message는 이런 것이다 >> input(*"입력:"*))
2. 개행문자를 그대로 받아서 반환한다.
