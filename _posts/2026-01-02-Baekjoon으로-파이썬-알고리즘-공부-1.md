## 1.입출력과 사칙연산

기본적인 print와 사칙연산 문법을 다루는 문제들로 이루어져있다. 파이썬을 공부해본 사람이라면 별로 어렵지 않게 문제를 풀 수 있을 것이다.

**입력값 받는 방법**
기본적으로 input()을 통해 한 줄의 입력값을 받을 수 있지만, ex 5 3 이런 경우의 입력값을 한 번에 변수 2개로 나누어 받는 방법이다
{% highlight js %}
// input을 공백 기준 split 해서 int 타입으로 각각 변수에 mapping 
a, b = map(int, input.split())
{% endhighlight %}

**사칙연산 기호**
{% highlight js %}
// 더하기
a + b

// 빼기
a - b

// 곱하기
a * b

// 나누기 (소수점 포함)
a / b

// 나누기 (몫만)
a // b

// 나누기 (나머지만)
a % b
{% endhighlight %}

**print 사용하기**
{% highlight python %}
// 변수가 str 타입일 경우 + 로 이어 붙여서 print
a = "am"
print("I " + a + " happy")

// "" 안에서 {} 안에 변수명 넣어 문자열 방식으로 한 번에 print
print(f"I {a} happy")
{% endhighlight %}
